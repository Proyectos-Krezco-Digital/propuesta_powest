# Skill: Sales Page Copywriting (Yashaiguy-dev Structure)
## Descripción
Estructura profesional para landing pages de venta de alto impacto.

## Fases de Estructura
### 1. Captura de ADN de Producto
- Identificar posición, audiencia y diferenciación.

### 2. Secciones Críticas
- **Hero:** Titular, Subtitular, CTA y Dirección Visual.
- **Agitación del Problema:** 3-4 puntos de dolor específicos.
- **Introducción de Solución:** Cómo funciona en 3-4 pasos simples.
- **Beneficios:** 3 beneficios de transformación con micro-prueba.
- **Manejo de Objeciones:** FAQs pre-emptivas.

### 3. Validación Hostil
- Probar el texto contra personas escépticas para cerrar brechas de credibilidad.
