# Skill: Classic Copywriting Formulas
## Descripción
Uso de estructuras probadas como AIDA y PAS para ventas directas.

## Fórmulas
### AIDA (Atención, Interés, Deseo, Acción)
1. **Atención:** Gancho potente o beneficio inmediato.
2. **Interés:** Datos o historias que enganchen.
3. **Deseo:** Mostrar la transformación del usuario con el producto.
4. **Action:** Instrucción clara de compra.

### PAS (Problema, Agitación, Solución)
1. **Problema:** El dolor del atleta (ropa incómoda, baja resistencia).
2. **Agitación:** Las consecuencias de no mejorar (lesiones, frustración).
3. **Solución:** FSO como la herramienta de superación.
